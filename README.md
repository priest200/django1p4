created by Tyron Njuguna 

## Description
Neighborhood is a web application that allows users to know the where abouts of their Hood

## Technologies Used
The following technologies have been used on this project:

* HTML
* CSS
* JS
* Bootstrap
* Django
* Python

## Setup / Installation
clone the repo:
* git clone https://github.com/priest200/django1p4.git
* cd neighborhood
* create virtual environment
* python3 -m venv venv
To activate the virtual environment
* source venv/bin/activate
install the packages from requirements.txt
* pip install -r requirements.txt
* setup environment variables
* cp .env.example .env
* Start the server
* python3.8 manage.py runserver
* open the browser and navigate to http://127.0.0.1:8000/ to see the application in action
